## 功能增强 (Features)

- **req 插件**：`/req:review-pr` 无阻塞时支持审核通过/合并/不处理 ([9305832](https://github.com/zhouhao4221/devflow-claude/commit/9305832))
  - 审查发现无风险（无阻塞问题）时，PR 仍在审核中，新增操作选择菜单：
    - [1] 审核通过 — 在 GitHub/Gitea 平台标记 PR 为已批准
    - [2] 审核并合并 — 批准 + 立即合并
    - [3] 不处理 — 保留当前状态
